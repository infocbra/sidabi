Descrição Geral

Este módulo tem o objetivo de gerenciar os dados de coleta resultantes de sessões de coletas com participantes e/ou pacientes com a doença de Parkinson. Este controle organizado e seguro das informações permite o acompanhamento, ao longo do tempo, de exames dos pacientes, auxilia no controle das linhas de pesquisa, além de evitar a fragmentação e perda de dados. O módulo denominado Biodata foi desenvolvido com base em requisitos levantados pelos questionamentos e discussões com alunos do laboratório NIATS, bem como pelo Prof. Adriano Adrande.
Toda a codificação client-side e server-side foi realizada por João Paulo Folador.

Orientador
Prof. Adriano O. Andrade, PhD

Ambiente de desenvolvimento

Client-side

Para a criação do lado do cliente, a aplicação web foi desenvolvida utilizado a linguagem de formatação HTML5, a linguagem de folhas de estilo CSS3, a biblioteca de funções em javascript para manipulação de eventos JQuery 3.1.1, o framework front-end Bootstrap 3.3.7 para adequar a responsividade em diferentes plataformas, dentre outras ferramentas.

Server-side

No back-end foi utilizado a linguagem de programção web PHP 5.6 junto com banco de dados postgresql 9.5 .
