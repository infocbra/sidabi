PRODATA

Este módulo de pesquisa de dados tem o objetivo de facilitar o entendimento das informações armazenadas em todo o sistema. Neste sentido, ele permite buscar os dados cadastros, por meio de filtros previamente escolhidos, a fim de listar todas as informações de um dado paciente/participante.

Outras ferramentas serão implementadas nesta área, objetivando auxiliar a tomada de decisão dos profissionais envolvidos.
