Esta área de Gerencimanto possibilita ao usuário do software: inserir, atualizar e apagar informações relacionadas ao controle de dados dos sintomas da doenças de Parkinson. Deste modo, no menu vertical lateral encontram-se as opções necessárias para gerenciar os dados.

Categorias: nesta opção, o usuário poderá gerenciar as categorias que agrupam os sintomas. É possível inserir, editar ou apagar uma categoria. É necessário que a categoria seja cadastrada com o campo 'Status' ativo, para que ela possa ser utilizada.

Sintomas: refere-se ao cadastro dos sintomas que comporão o quadro de uma dada categoria. Nesta opção é possível inserir, apagar ou editar um sintoma da doença de Parkinson.
